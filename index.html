Projects

Improving Face Perceptual Quality in Learned Image Compression

2026

Studies face quality in machine learning-based image compression by optimizing Wasserstein Distortion as a perceptual objective, guiding neural codecs to better preserve facial details under low-bitrate settings.

Explainable Image Aesthetic Factor Ranking

2025

Applies machine learning to model human aesthetic perception from interpretable visual factors, using the AVA dataset to predict aesthetic ratings and rank perceptual cues that influence perceived image quality.

Real-Time Music-Driven Visual Modulation on Static Image

2025

Explores multimodal perception by mapping real-time music features to smooth visual modulations on static images, creating synchronized audio-visual effects that enhance human perceptual experience.

Accelerating the Design of Perovskite Solar Cells Using Active Learning

2023

Uses machine learning and active learning to accelerate perovskite material design through predictive modeling. Applies data-driven iteration to guide material design decisions.
