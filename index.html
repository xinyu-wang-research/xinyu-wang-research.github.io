Projects

Improving Face Perceptual Quality in Learned Image Compression

2026

Studies how machine learning-based image compression models can better preserve face quality under low-bitrate settings. This project optimizes Wasserstein Distortion as a perceptual objective to guide neural image compression toward reconstructions that better match human perception of facial details.

Explainable Image Aesthetic Factor Ranking

2025

Applies machine learning to study human aesthetic perception by predicting aesthetic ratings from interpretable visual factors. Using the AVA dataset, this project ranks perceptual cues to analyze how different visual properties contribute to perceived image quality and aesthetic preference.

Real-Time Music-Driven Visual Modulation on Static Image

2025

Explores multimodal perception through real-time audio-driven visual modulation. This project maps music features to calibrated visual control parameters on static images, creating synchronized audio-visual effects that enhance human perceptual experience and engagement.

Accelerating the Design of Perovskite Solar Cells Using Active Learning

2023

Uses machine learning and active learning to guide perovskite solar cell material design. This project develops predictive models for material performance and applies data-driven iteration to select promising candidates more efficiently than exhaustive search.
