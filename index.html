Projects

Explainable Image Aesthetic Factor Ranking

2025

Predicts 10-bin human rating distributions (1–10) from interpretable, editing-aligned visual factors, and ranks factor importance via ablation. Frames the model as explainable representation learning for lightweight human response modeling.

Real-Time Music-Driven Visual Modulation on Static Image

2025

A low-latency audio-to-visual system that translates music dynamics into smooth visual effects, to improve perceived immersion and engagement. It links real-time audio features to visual control parameters through a calibrated, smoothed mapping.

Optimal Modulation of Interleukin-8 During Surgery Using Repertaxin

2025

Modeled and controlled intraoperative inflammation by optimizing Repertaxin dosing with state estimation.

  • Built a state-space model + feedback controller to modulate IL-8 levels during surgery.
  • Designed a Kalman filter estimator and personalized dosing policy; validated on 12-hour, 10-patient cardiac-surgery IL-8 time series in MATLAB/Simulink.

Accelerating the Design of Perovskite Solar Cells Using Active Learning

2023

Used machine learning + active learning ideas to accelerate perovskite material design through predictive modeling. Built and tuned models (e.g., SVM/XGBoost) and used data-driven iteration to guide material design decisions.

Balls on an Elastic Band

2022

Investigated two ball-on-rubber-band phenomena (including equipotential-line behavior) via experimental design, modeling, and analysis.
